Need help integrating or debugging Interhuman? Use the options below so you can get unblocked quickly.

## Join the community

Ask questions and learn from other builders in the Interhuman community:

- [community.interhuman.ai](https://community.interhuman.ai)

## Contact Interhuman support

If you need direct help from our team, email:

- <a href="mailto:support@interhuman.ai">support@interhuman.ai</a>

When you report an issue, please include the `correlation_id` and `error_id` if you have them, the endpoint you called, the approximate timestamp and timezone, and a short note on what you expected to happen versus what actually happened.

You can find `error_id` and `correlation_id` in Interhuman API error responses. See [Error handling](/api-reference/error-handling) for the response format and troubleshooting details.
